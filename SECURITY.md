# Security Policy

## Reporting a Vulnerability

Please report suspected security vulnerabilities privately through GitHub Security Advisories for this repository.

Do not open a public issue for a security vulnerability. Include:

- affected AOSC version or commit
- affected OpenSearch version
- reproduction steps or proof of concept
- expected impact
- any known mitigations

Maintainers will acknowledge reports on a best-effort basis and coordinate next steps through the private advisory.

## Supported Versions

Security fixes are targeted at supported release lines listed in the project documentation and GitHub releases.
